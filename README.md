1. analysis_utils.py (статистика + графики)
Что делает:

грузит CSV и приводит числа к numeric;

считает статистику;

строит 3 графика Matplotlib;

возвращает картинки в base64 (удобно отдавать как PNG через Flask).

2. app.py (главный сервер Flask)
   Что делает app.py:

загружает данные (через analysis_utils.df);

принимает фильтры из URL и фильтрует DataFrame;

отдаёт страницы index, favorites, recommend;

отдаёт графики /plot/... как PNG;

делает favorites через session;

делает экспорт /export;

делает подсказки поиска /api/search.

3. templates/base.html (общий каркас)
   единый дизайн для всех страниц.

4. templates/index.html (главная: фильтры + таблица + графики + подсказки)
   За что отвечает index.html:

форма фильтров (GET → /);

таблица результатов;

кнопки: применить / сбросить / экспорт;

звёздочка → добавление в избранное;

“марка→модель” через JS;

подсказки поиска через /api/search.

5. templates/favorites.html
   Что делает: показывает все id, сохранённые в session["favorites"].

6. templates/recommend.html
   Что делает: показывает 2 группы рекомендаций из app.py.

7. static/style.css (единый стиль + “красиво”)
   Что делает: единый “тёмный” дизайн, анимации появления карточек, красивые кнопки, таблица, подсказки поиска.
